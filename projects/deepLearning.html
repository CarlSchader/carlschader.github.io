FlexNet
Description

FlexNet is my most recent framework for creating flexible generalized neural networks for deep learning. It allows the programmer to create a net object whose constructor allows them to specify the number of layers, node count of each layer, type of neural network, what type of activation functions are used, and the learning rate. The networks created by FlexNet are completely vectorized and extremely performant compared to my last project Neuron. The framework is also very user friendly allowing programmers to quickly create neural networks without any knowledge of the math behind the scenes. forward propogation and backpropogation are methods on the object. Just pass in your entire data set and begin!

Neuron
Description

Neuron is my first attempt at creating a library to generalize the creation of neural networks for deep learning. It allows the programmer using the framework to create a neuralNet object with as many layers of any size they want, allowing them to easily train/test their networks and save/load them too. Creating this framework was how I taught myself the mathematics behind deep learning and how to implement it in code in a generalized way. Neuron is not intuitive to use for someone without documentation. The math and code is also not fully vectorized and uses sigmoid functions as the only activation function (which is considered a poor activation function). I recommend using my newer and much faster vectorized framework flexNet which can be found above.
